# Agno (agno)

Agno (formerly Phidata) is a high-performance open-source Python framework for building multi-modal, multi-agent systems with memory, knowledge, tools, and reasoning capabilities. It provides both an agent development SDK and AgentOS, a production runtime that transforms agents into managed APIs with 50+ endpoints covering runs, sessions, memory, knowledge, traces, evaluations, and scheduling. AgentOS supports RBAC, per-user session isolation, streaming via Server-Sent Events, background execution, and human-in-the-loop approval flows across 30+ LLM providers.

## APIs

### AgentOS Agent API

The AgentOS Agent API provides 50+ RESTful endpoints for managing agents in production. Endpoints cover runs, sessions, memory, knowledge, evaluations, traces, metrics, schedules, and approvals. Authentication uses JWT Bearer tokens with RBAC. The API is FastAPI-based with live docs at /docs and an OpenAPI spec at /openapi.json.

- Documentation: [https://docs.agno.com/features/api](https://docs.agno.com/features/api)
